
📊 Interactive Dashboard Development
📌 Overview

This project creates an interactive dashboard to display key insights like sales, profit, and trends using charts and filters.

🛠️ Tools Used
Python
Streamlit
Pandas
Plotly
🚀 How to Run
pip install streamlit pandas plotly
streamlit run app.py
📈 Features
Interactive filters (Region, Product)
Sales & Profit metrics
Bar and Line charts
📂 Files
app.py → Dashboard code
data.csv → Dataset
